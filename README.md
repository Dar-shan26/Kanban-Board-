# Kanban Task Management Board

A modern and responsive Kanban Task Management Board built using **HTML, CSS, and JavaScript**.

## Features

* Drag and Drop Tasks
* Add New Tasks
* Delete Tasks
* Task Counter
* Responsive Design
* LocalStorage Support
* Clean and Modern UI

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Structure

```
├── index.html
├── style.css
├── script.js
└── assets/
```

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
```

2. Open the project folder.

3. Open `index.html` in your browser.
